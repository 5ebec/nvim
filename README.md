# my nvim
## usage
1. Install neovim nightly  
https://github.com/neovim/neovim/wiki/Installing-Neovim  
https://github.com/neovim/neovim/releases/nightly  
1. Install pynvim
  ```shell
  $ pip install pynvim
  ```
1. Clone this repo
  ```shell
  $ git clone https://github.com/5ebec/nvim ~/.config/nvim
  ```
